# Artifacts: Enriching Experiment Artifacts with LLMs

This repository contains the evaluation artifacts for the short paper **"Enriching Experiment Artifacts with LLMs for Comprehensive and Understandable Documentation"**. 
The paper was authored by Kilian Warmuth, Eric Hauser, Daniel Petri, Marcel Kempf, and Georg Carle. It has been accepted at the 2026 ACM Conference on Reproducibility and Replicability (ACM REP'26).

The artifact data is available via GitHub Pages for the following two experiments:
- [Artifacts of the NVIDIA Spark](https://github.com/tumi8/paper-llm-artifacts-ai-enrichments-spark)
- [Artifacts of the GPU Server](https://github.com/tumi8/paper-llm-artifacts-ai-enrichments-gpu-server)
